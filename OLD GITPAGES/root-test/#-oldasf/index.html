Fonctionnalités

Navigation hiérarchique

Le système gère automatiquement une navigation à trois niveaux :

  • Menu principal : Sections principales (Projets, Créations, Idées)
  • Pages showcase : Galeries de vos projets par catégorie
  • Pages détaillées : Contenu complet avec table des matières

Ce système vous permet de vous concentrer sur votre contenu plutôt que sur le code HTML.

Styles Markdown avancés

Support complet de la syntaxe Markdown avec des styles modernes :

// Exemple de code JavaScript
function parseMarkdown(content) {
    return marked.parse(content, {
        highlight: function(code, lang) {
            return hljs.highlight(code, {language: lang}).value;
        }
    });
}
